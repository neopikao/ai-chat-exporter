# AI Chat Exporter – Official Site

GitHub Pages site for [AI Chat Exporter](https://chromewebstore.google.com/detail/ai-chat-exporter/loodiigkfmmieeaiililegmhjgjjlcaj), a Chrome Extension that exports conversations from 8 AI chat services (Gemini, ChatGPT, Claude, Grok, Genspark, Microsoft Copilot, DeepSeek, Perplexity) as Markdown.

## Pages

| File | URL |
|------|-----|
| `index.html` | App introduction page |
| `privacy-policy.html` | Privacy Policy (EN / JA) |

## Setup

1. Push this folder to a GitHub repository (e.g. `ai-chat-exporter`)
2. Go to **Settings → Pages → Source: Deploy from branch (main / root)**
3. Site will be live at `https://<your-username>.github.io/ai-chat-exporter/`
