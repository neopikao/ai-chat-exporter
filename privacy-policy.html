Privacy Policy

Last updated: 2026-03-17

English

1. Data Collection

The Extension does NOT collect, store, transmit, or share any personal data or user information to the developer or any unauthorized third party.

2. Data Processing

All chat content is processed entirely within your browser:

  • Copied text is stored only in your local clipboard.
  • Downloaded files are saved only to your local device's download folder.
  • No data ever leaves your device unless you explicitly use a Pro feature (Google Docs / Notion).

3. Pro Version Features and External Services

  • Google Docs / Drive Integration — uses OAuth 2.0 via the Chrome identity API. Your data stays in your own Google Drive only and is never accessed by the developer.
  • Notion Integration — uses a user-provided Integration Token stored locally in chrome.storage.sync. This token is never transmitted to any external server other than Notion's official API.
  • Both features are activated only on your explicit action and are entirely optional.

4. Permissions Used

  • activeTab — to read the current tab's chat content.
  • downloads — to save Markdown files to your local folder.
  • storage — to save your settings locally (locale, toggle preferences, folder IDs).
  • identity — (Pro) to authenticate with Google via OAuth 2.0.
  • host_permissions — limited to the 8 supported AI chat services: Gemini, ChatGPT, Claude, Grok, Genspark, Microsoft Copilot, DeepSeek, and Perplexity.

5. Third-Party Services

This Extension does not use any analytics, advertising, or tracking services. The only external connections are user-initiated integrations with Google (OAuth 2.0) and Notion (official API).

6. Changes to This Policy

Any changes to this Privacy Policy will be reflected in an updated version of this document with a revised "Last updated" date.

7. Contact

If you have any questions or concerns about this Privacy Policy, please contact:

日本語

1. データの収集について

本拡張機能は、ユーザーの個人情報や会話データを開発者または未承認の第三者に収集・保存・送信・共有しません。

2. データの処理について

すべてのチャットコンテンツはブラウザ内でのみ処理されます：

  • コピーされたテキストはローカルのクリップボードにのみ保存されます。
  • ダウンロードされたファイルはローカルデバイスのダウンロードフォルダにのみ保存されます。
  • Pro機能（Googleドキュメント・Notion）を明示的に使用する場合を除き、データが端末外に送出されることはありません。

3. Pro版機能と外部サービスについて

  • Google ドキュメント / ドライブ連携 — Chrome identity API を通じてOAuth 2.0で認証します。データはユーザー自身のGoogle Driveにのみ保存され、開発者がアクセスすることはありません。
  • Notion連携 — ユーザーが提供したIntegration Tokenを chrome.storage.sync にローカル保存します。このトークンはNotionの公式API以外の外部サーバーに送信されることはありません。
  • いずれの機能もユーザーの明示的な操作によってのみ有効化される任意機能です。

4. 使用する権限について

  • activeTab — 現在のタブのチャットコンテンツを読み取るため。
  • downloads — Markdownファイルをローカルフォルダに保存するため。
  • storage — 設定（言語、トグル設定、フォルダIDなど）をローカルに保存するため。
  • identity — （Pro版）OAuth 2.0でGoogleに認証するため。
  • host_permissions — 対応する8つのAIチャットサービスのみに限定：Gemini、ChatGPT、Claude、Grok、Genspark、Microsoft Copilot、DeepSeek、Perplexity。

5. 第三者サービスについて

本拡張機能はアナリティクス・広告・トラッキングサービスを一切使用しません。外部への接続は、ユーザーが明示的に開始したGoogleとの連携（OAuth 2.0）およびNotionとの連携（公式API）のみです。

6. プライバシーポリシーの変更について

本プライバシーポリシーの変更は、「最終更新日」を更新した新しいバージョンとしてこのページに反映されます。
